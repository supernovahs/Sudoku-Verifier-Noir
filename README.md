# Sudoku-Verifier-Noir
A 4x4 sudoku verifier in Noir language


## Noir [Docs](https://noir-lang.org/)

## Install Noir

[Link](https://noir-lang.org/getting_started/nargo/nargo_installation)

## Compile

```
nargo check
```

## Proofs
```
nargo prove
```

## Verify

```
nargo verify
```


### To log for debugging

```
nargo prove --show-outputs

```
